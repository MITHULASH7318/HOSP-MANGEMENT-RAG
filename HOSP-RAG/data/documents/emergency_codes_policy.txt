HOSPITAL POLICY: EMERGENCY CODES AND SAFETY PROCEDURES
Document ID: POL-SAFE-002
Version: 2.5 | Effective Date: January 2024
Department: All Departments | Owner: Safety & Security Committee

1. EMERGENCY CODE SYSTEM

CODE RED - FIRE
Activation: Pull nearest fire alarm AND call ext. 3333
RACE Protocol:
- R = RESCUE: Remove patients from immediate danger
- A = ALARM: Activate fire alarm, call 3333
- C = CONTAIN: Close all doors to isolate fire
- E = EXTINGUISH/EVACUATE: Use extinguisher if safe; evacuate otherwise
PASS Protocol for extinguishers:
- P = PULL pin
- A = AIM at base of fire
- S = SQUEEZE handle
- S = SWEEP side to side
Evacuation: Horizontal first → Vertical (stairwells only, never elevators)

CODE BLUE - CARDIAC/RESPIRATORY ARREST
- Call ext. 3333 and announce Code Blue + location
- Begin CPR immediately (if trained)
- Get AED – located at nursing stations and main corridors
- Assign roles: Compressor, Airway, Recorder, IV Access, Medications
- Compression rate: 100-120/min, depth: 2-2.4 inches
- Ratio: 30 compressions : 2 breaths (until advanced airway placed)
- Switch compressors every 2 minutes
- Document all interventions with timestamps

CODE PINK - INFANT/CHILD ABDUCTION
- Call 3333 immediately
- Do NOT approach suspect alone
- Secure all exits immediately
- No one enters or exits facility
- Staff at entrances request photo ID from all persons leaving
- Give full description of suspect to security

CODE GRAY - VIOLENT/COMBATIVE PERSON
- Call 3333 for security
- Do not intervene physically unless trained
- Move other patients away from area
- Attempt verbal de-escalation only if safe
- Document incident in RL Solutions

CODE ORANGE - HAZARDOUS MATERIAL SPILL
- Evacuate area immediately
- Call 3333 and identify material if known
- Consult Safety Data Sheet (SDS) – available in each department
- Decontaminate exposed persons before treatment
- Do not re-enter until cleared by HazMat team

CODE WHITE - EXTERNAL DISASTER/MASS CASUALTY
- Hospital Incident Command System (HICS) activated
- Report to your department head for assignment
- Discharge/transfer patients to create capacity
- Cancel elective procedures
- Clinical staff report to designated surge areas

CODE PURPLE - BOMB THREAT
- Do not hang up on caller; try to get details (what kind of bomb, when, where)
- Call 3333 from another line
- Do not use personal cell phones or radio in the area
- Security and administration decide on evacuation

2. FIRE SAFETY
- Keep all corridors and exits clear at all times
- Fire doors must remain closed (never propped open)
- No combustibles stored in corridors
- Annual fire safety training mandatory for all staff
- Know location of nearest fire extinguisher, pull station, and exit

3. DISASTER/EMERGENCY PREPAREDNESS
- All staff must complete annual emergency preparedness training
- Department-specific emergency plans posted in each unit
- Hospital Emergency Operations Plan available on intranet

4. WORKPLACE SAFETY
- Report all workplace injuries to supervisor AND occupational health immediately
- Ergonomic training required for patient handling
- Use mechanical lifts for patients over 35 lbs (unassisted patient handling)
- Safety Data Sheets for all chemicals available in department binders and online

5. INFANT SECURITY
- All infants wear electronic security bands
- Never remove infant from unit without mother's verified wristband match
- Alarm sounds at elevator/exit if band approaches
- All staff wear photo ID; report anyone without ID near nursery
